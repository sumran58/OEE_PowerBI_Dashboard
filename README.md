# OEE Power BI Dashboard

## Project Overview
This project analyzes the Overall Equipment Effectiveness (OEE) of production lines using Power BI. It includes visualizations of production, downtime, and quality losses.

## Features
- Interactive dashboard with trend lines, slicers, and filters.
- OEE calculation including Availability, Performance, and Quality metrics.
- Downtime analysis by reason and category.
- Quality loss analysis with Pareto charts.

## Dataset
- `OEE_Data.xlsx` → Contains production line data including planned time, run time, good/scrap counts.
- `Downtime_Reasons.xlsx` → Contains downtime reasons, durations, and categories.
- `Quality_Loss.xlsx` → Contains defect reasons and quantities.

## Tools Used
- Power BI Desktop
- Microsoft Excel

## How to Use
1. Download the repository.
2. Open `OEE_Dashboard.pbix` in Power BI Desktop.
3. Explore the interactive dashboard and filters.

## Insights / Results
- Identified top downtime reasons and their impact on OEE.
- Visualized trends in production, scrap, and quality losses.
